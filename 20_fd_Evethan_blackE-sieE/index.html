Setting Up


To use Foundation CDN in your HTML, copy and paste the following as your stylesheet in your index.html file:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.7.5/dist/css/foundation.min.css" crossorigin="anonymous">

You can also create another stylesheet for basic stylistic changes, like background and text color:

Run this in terminal to create the style-file!

$touch style.css

Paste this code in your HTML file!

<link rel="stylesheet" href="style.css">

Implementing Cards


In this section, we create dynamic, responsive cards using Foundation's grid system. Each card is created inside a div with the class card, and we use the Foundation grid system to ensure that the cards are responsive and adjust to different screen sizes.

The Foundation grid classes used:

cell: Creates a grid item (used for each card). small-12 medium-4: Specifies how much space the card will occupy on small and medium screens. grid-x: Creates a horizontal grid layout. text-center: Centers the content (used for card title). card: Styles the div as a card with a shadow and structure. card-divider: Creates the divider for the title of the card. card-section: Defines the content area of the card.

Card Code Example:

<div class="grid-container">
<div class="grid-x align-center">
<div class="cell small-12 medium-4">
<div class="card" style="width: 300px;">
<div class="card-divider text-center">Foundation Astronaut Yeti</div>
<img src="imgs/img1.png" alt="Astronaut Yeti">
<div class="card-section">
<h4>Card 1</h4>
<p>Lorem ipsum dolor sit amet...</p>
</div> </div> </div> </div> </div>



Implementing Upload and Display


This section allows the user to upload images and display them dynamically on the webpage. We use an HTML file input for selecting the images and JavaScript to handle the display.

The Foundation classes used:

grid-container: A container for the grid layout. grid-x: Creates a horizontal grid layout for the display area. grid-margin-x: Adds margin between the grid items. small-up-2 medium-up-3 large-up-4: Specifies the number of columns for small, medium, and large screens for displaying images. button: Styles the submit button for uploading images.

Upload and Display Code Example:

<form id="uploadForm">
<label>Choose images to upload:</label>
<input type="file" id="imageInput" multiple accept="image/*">
<button type="submit" class="button">Upload Images</button>
</form>
<div id="imageDisplay" class="grid-x grid-margin-x small-up-2 medium-up-3 large-up-4"></div>
